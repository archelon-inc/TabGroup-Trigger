## Подробное описание

```
Расширение Chrome, которое обеспечивает переключение групп вкладок на основе сочетаний клавиш с использованием вызова URL. Объедините это расширение с утилитами горячих клавиш, чтобы мгновенно переключаться между группами вкладок Chrome, используя ваши любимые горячие клавиши.

【Зачем это расширение?】
Chrome не предоставляет встроенных сочетаний клавиш для переключения между группами вкладок. Это расширение решает эту проблему, позволяя вам запускать переключение групп вкладок через URL-адреса, которые затем могут быть сопоставлены с сочетаниями клавиш с помощью внешних утилит. Например, нажмите Cmd+1, чтобы перейти к группе "Работа", Cmd+2 для "Личное" и так далее.

【Основные функции】
✓ Интеграция с горячими клавишами
  Бесшовно работает с утилитами горячих клавиш для обеспечения переключения групп вкладок на основе сочетаний клавиш

✓ Запуск на основе URL
  Перейдите по адресу https://extension.tabgroup-trigger/{имя-группы}, чтобы мгновенно переключиться на указанную группу вкладок

✓ Всплывающее окно со списком URL
  Нажмите на значок расширения, чтобы просмотреть все группы вкладок с их URL-кодированными адресами, легко копируемыми одним щелчком

【Как использовать】
1. Создайте группы вкладок в Chrome и дайте им имена
2. Нажмите на значок расширения, чтобы получить URL для каждой группы:
   - URL-кодированные адреса отображаются автоматически
   - Нажмите кнопку "Копировать URL", чтобы скопировать в буфер обмена
   - Нажмите на область отображения URL, чтобы выбрать текст для ручного копирования
3. Настройте действия открытия URL в вашей утилите горячих клавиш
   Пример: Cmd+Shift+1 → https://extension.tabgroup-trigger/Work
4. Нажмите настроенную горячую клавишу, чтобы мгновенно переключить группу вкладок

【Примеры утилит горячих клавиш】
• macOS: BetterTouchTool, Keyboard Maestro
• Windows: AutoHotkey, PowerToys (версия 0.79 или выше)

【Пример настройки (BetterTouchTool)】
1. Создайте новое сочетание клавиш (например, Cmd+Shift+1)
2. Установите действие "Open URL"
3. Введите URL: https://extension.tabgroup-trigger/Work
4. Теперь нажатие Cmd+Shift+1 мгновенно переключит на вашу группу вкладок "Work"

【Советы】
• Расширение запоминает, какая вкладка была последней активной в каждой группе
• Когда вы переключаетесь на группу, оно активирует вкладку, которую вы просматривали последней
• Если истории вкладок не существует, оно активирует первую вкладку в группе

【Ограничения】
• Невозможно переключиться на сохраненные группы вкладок (функция Chrome "Сохранить группу", которая удаляет все вкладки, но сохраняет метку группы)
  Примечание: Свернутые группы вкладок (где вкладки скрыты, но не удалены) работают нормально
• Специальные символы в именах групп (?, /, #, %, и т.д.) требуют URL-кодирования
  Пример: "Question?Answer" → https://extension.tabgroup-trigger/Question%3FAnswer
  Используйте всплывающий интерфейс расширения для автоматического получения URL-кодированных адресов
• Если вы выполните 25+ последовательных переключений групп вкладок, более старые вкладки могут быть невосстановимы через Cmd+Shift+T из-за ограничения истории сеансов Chrome (chrome.sessions.MAX_SESSION_RESULTS = 25)

【Конфиденциальность】
Это расширение не собирает никаких данных. Вся обработка выполняется локально на вашем устройстве.
```
