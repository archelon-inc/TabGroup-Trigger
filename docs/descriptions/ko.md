# Chrome 웹 스토어 설명 (한국어)

## 짧은 설명 (132자 이내)

```
URL 호출을 통해 Chrome 탭 그룹을 전환합니다. 키보드 단축키 유틸리티와 결합하여 핫키 기반 즉시 그룹 전환이 가능합니다.
```

문자 수: 78자

## 상세 설명

```
URL 호출을 사용하여 키보드 단축키 기반 탭 그룹 전환을 가능하게 하는 Chrome 확장 프로그램입니다. 이 확장 프로그램을 키보드 단축키 유틸리티와 결합하여 좋아하는 핫키를 사용해 Chrome 탭 그룹 간에 즉시 전환할 수 있습니다.

【왜 이 확장 프로그램이 필요한가요?】
Chrome은 탭 그룹 간 전환을 위한 기본 키보드 단축키를 제공하지 않습니다. 이 확장 프로그램은 URL을 통해 탭 그룹 전환을 트리거할 수 있도록 하여 이 문제를 해결하며, 외부 유틸리티를 사용하여 키보드 단축키에 매핑할 수 있습니다. 예를 들어, Cmd+1을 눌러 "업무" 그룹으로 이동하고, Cmd+2를 눌러 "개인" 그룹으로 이동하는 등의 작업이 가능합니다.

【주요 기능】
✓ 키보드 단축키 통합
  키보드 단축키 유틸리티와 원활하게 작동하여 핫키 기반 탭 그룹 전환 제공

✓ URL 기반 트리거
  https://extension.tabgroup-trigger/{그룹명} 으로 이동하여 지정된 탭 그룹으로 즉시 전환

✓ URL 목록 팝업
  확장 프로그램 아이콘을 클릭하여 URL 인코딩된 URL과 함께 모든 탭 그룹을 확인하고, 한 번의 클릭으로 쉽게 복사

【사용 방법】
1. Chrome에서 탭 그룹을 만들고 이름을 지정합니다
2. 확장 프로그램 아이콘을 클릭하여 각 그룹의 URL을 가져옵니다:
   - URL 인코딩된 URL이 자동으로 표시됩니다
   - "URL 복사" 버튼을 클릭하여 클립보드에 복사
   - URL 표시 영역을 클릭하여 수동 복사를 위한 텍스트 선택
3. 키보드 단축키 유틸리티에서 URL 열기 작업을 설정합니다
   예시: Cmd+Shift+1 → https://extension.tabgroup-trigger/Work
4. 구성된 핫키를 눌러 탭 그룹을 즉시 전환합니다

【키보드 단축키 유틸리티 예시】
• macOS: BetterTouchTool, Keyboard Maestro
• Windows: AutoHotkey, PowerToys (버전 0.79 이상)

【설정 예시 (BetterTouchTool)】
1. 새 키보드 단축키 생성 (예: Cmd+Shift+1)
2. 작업을 "Open URL"로 설정
3. URL 입력: https://extension.tabgroup-trigger/Work
4. 이제 Cmd+Shift+1을 누르면 "Work" 탭 그룹으로 즉시 전환됩니다

【팁】
• 확장 프로그램은 각 그룹에서 마지막으로 활성화된 탭을 기억합니다
• 그룹으로 전환하면 마지막으로 보고 있던 탭이 활성화됩니다
• 탭 기록이 없으면 그룹의 첫 번째 탭이 활성화됩니다

【제한 사항】
• 저장된 탭 그룹으로 전환할 수 없습니다 (Chrome의 "그룹 저장" 기능은 모든 탭을 제거하지만 그룹 레이블은 유지합니다)
  참고: 축소된 탭 그룹 (탭이 숨겨져 있지만 제거되지 않은 경우)은 정상적으로 작동합니다
• 그룹 이름의 특수 문자 (?, /, #, % 등)는 URL 인코딩이 필요합니다
  예시: "Question?Answer" → https://extension.tabgroup-trigger/Question%3FAnswer
  확장 프로그램의 팝업 UI를 사용하여 자동으로 URL 인코딩된 URL 가져오기
• 25회 이상의 연속 탭 그룹 전환을 수행하면 Chrome의 세션 기록 제한 (chrome.sessions.MAX_SESSION_RESULTS = 25)으로 인해 이전 탭은 Cmd+Shift+T를 통해 복원할 수 없을 수 있습니다
```
