# وصف متجر Chrome الإلكتروني (العربية)

## الوصف المختصر (132 حرفًا كحد أقصى)

```
تبديل مجموعات علامات تبويب Chrome عبر استدعاء URL. ادمجها مع أدوات اختصارات لوحة المفاتيح للتبديل الفوري بمفاتيح الاختصار.
```

عدد الأحرف: 126 حرفًا

## الوصف التفصيلي

```
امتداد Chrome يتيح تبديل مجموعات علامات التبويب بناءً على اختصارات لوحة المفاتيح باستخدام استدعاء URL. ادمج هذا الامتداد مع أدوات اختصارات لوحة المفاتيح للتبديل الفوري بين مجموعات علامات تبويب Chrome باستخدام مفاتيح الاختصار المفضلة لديك.

【لماذا هذا الامتداد؟】
لا يوفر Chrome اختصارات لوحة مفاتيح أصلية للتبديل بين مجموعات علامات التبويب. يحل هذا الامتداد هذه المشكلة من خلال السماح لك بتشغيل تبديل مجموعات علامات التبويب عبر عناوين URL، والتي يمكن بعد ذلك ربطها باختصارات لوحة المفاتيح باستخدام أدوات خارجية. على سبيل المثال، اضغط على Cmd+1 للانتقال إلى مجموعة "العمل"، Cmd+2 لـ "الشخصية"، وهكذا.

【الميزات الرئيسية】
✓ التكامل مع اختصارات لوحة المفاتيح
  يعمل بسلاسة مع أدوات اختصارات لوحة المفاتيح لتوفير تبديل مجموعات علامات التبويب بناءً على مفاتيح الاختصار

✓ التشغيل بناءً على URL
  انتقل إلى https://extension.tabgroup-trigger/{اسم-المجموعة} للتبديل الفوري إلى مجموعة علامات التبويب المحددة

✓ نافذة منبثقة لقائمة URL
  انقر على أيقونة الامتداد لعرض جميع مجموعات علامات التبويب مع عناوين URL المشفرة، يمكن نسخها بسهولة بنقرة واحدة

【كيفية الاستخدام】
1. أنشئ مجموعات علامات تبويب في Chrome وأعطها أسماء
2. انقر على أيقونة الامتداد للحصول على عناوين URL لكل مجموعة:
   - يتم عرض عناوين URL المشفرة تلقائيًا
   - انقر على زر "نسخ URL" للنسخ إلى الحافظة
   - انقر على منطقة عرض URL لتحديد النص للنسخ اليدوي
3. قم بإعداد إجراءات فتح URL في أداة اختصارات لوحة المفاتيح
   مثال: Cmd+Shift+1 → https://extension.tabgroup-trigger/Work
4. اضغط على مفتاح الاختصار المكوّن للتبديل الفوري لمجموعة علامات التبويب

【أمثلة على أدوات اختصارات لوحة المفاتيح】
• macOS: BetterTouchTool، Keyboard Maestro
• Windows: AutoHotkey، PowerToys (الإصدار 0.79 أو أحدث)

【مثال على الإعداد (BetterTouchTool)】
1. أنشئ اختصار لوحة مفاتيح جديد (مثل: Cmd+Shift+1)
2. اضبط الإجراء على "Open URL"
3. أدخل عنوان URL: https://extension.tabgroup-trigger/Work
4. الآن سيؤدي الضغط على Cmd+Shift+1 إلى التبديل الفوري إلى مجموعة علامات التبويب "Work"

【نصائح】
• يتذكر الامتداد علامة التبويب التي كانت نشطة آخر مرة في كل مجموعة
• عند التبديل إلى مجموعة، سينشط علامة التبويب التي كنت تعرضها آخر مرة
• إذا لم يكن هناك سجل لعلامات التبويب، فسينشط أول علامة تبويب في المجموعة

【القيود】
• لا يمكن التبديل إلى مجموعات علامات التبويب المحفوظة (ميزة "حفظ المجموعة" في Chrome التي تزيل جميع علامات التبويب لكن تحتفظ بتسمية المجموعة)
  ملاحظة: مجموعات علامات التبويب المطوية (حيث تكون علامات التبويب مخفية لكن غير محذوفة) تعمل بشكل جيد
• الأحرف الخاصة في أسماء المجموعات (؟، /، #، %، إلخ.) تتطلب تشفير URL
  مثال: "Question?Answer" → https://extension.tabgroup-trigger/Question%3FAnswer
  استخدم واجهة المستخدم المنبثقة للامتداد للحصول تلقائيًا على عناوين URL المشفرة
• إذا قمت بـ 25+ تبديل متتالي لمجموعات علامات التبويب، فقد لا يمكن استعادة علامات التبويب الأقدم عبر Cmd+Shift+T بسبب حد سجل الجلسة في Chrome (chrome.sessions.MAX_SESSION_RESULTS = 25)
```
