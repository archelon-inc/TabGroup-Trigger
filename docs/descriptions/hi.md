# Chrome वेब स्टोर विवरण (हिंदी)

## संक्षिप्त विवरण (132 वर्ण अधिकतम)

```
URL से टैब समूह स्विच करें। कीबोर्ड शॉर्टकट के साथ त्वरित स्विचिंग।
```

वर्ण संख्या: 69 वर्ण

## विस्तृत विवरण

```
एक Chrome एक्सटेंशन जो URL आह्वान का उपयोग करके कीबोर्ड शॉर्टकट-आधारित टैब समूह स्विचिंग को सक्षम करता है। इस एक्सटेंशन को कीबोर्ड शॉर्टकट उपयोगिताओं के साथ संयोजित करें ताकि आप अपनी पसंदीदा हॉटकीज़ का उपयोग करके Chrome टैब समूहों के बीच तुरंत स्विच कर सकें।

【यह एक्सटेंशन क्यों?】
Chrome टैब समूहों के बीच स्विच करने के लिए मूल कीबोर्ड शॉर्टकट प्रदान नहीं करता है। यह एक्सटेंशन उस समस्या को हल करता है जिससे आप URL के माध्यम से टैब समूह स्विच ट्रिगर कर सकते हैं, जिन्हें बाहरी उपयोगिताओं का उपयोग करके कीबोर्ड शॉर्टकट से मैप किया जा सकता है। उदाहरण के लिए, अपने "कार्य" समूह में जाने के लिए Cmd+1 दबाएं, "व्यक्तिगत" के लिए Cmd+2, और इसी तरह।

【मुख्य विशेषताएं】
✓ कीबोर्ड शॉर्टकट एकीकरण
  हॉटकी-आधारित टैब समूह स्विचिंग प्रदान करने के लिए कीबोर्ड शॉर्टकट उपयोगिताओं के साथ सहजता से काम करता है

✓ URL-आधारित ट्रिगरिंग
  निर्दिष्ट टैब समूह में तुरंत स्विच करने के लिए https://extension.tabgroup-trigger/{समूह-नाम} पर नेविगेट करें

✓ URL सूची पॉपअप
  एक्सटेंशन आइकन पर क्लिक करें सभी टैब समूहों को उनके URL-एन्कोडेड URL के साथ देखने के लिए, एक क्लिक से आसानी से कॉपी किया जा सकता है

【कैसे उपयोग करें】
1. Chrome में टैब समूह बनाएं और उन्हें नाम दें
2. प्रत्येक समूह के लिए URL प्राप्त करने के लिए एक्सटेंशन आइकन पर क्लिक करें:
   - URL-एन्कोडेड URL स्वचालित रूप से प्रदर्शित होते हैं
   - क्लिपबोर्ड पर कॉपी करने के लिए "URL कॉपी करें" बटन पर क्लिक करें
   - मैन्युअल कॉपी के लिए टेक्स्ट चुनने के लिए URL प्रदर्शन क्षेत्र पर क्लिक करें
3. अपनी कीबोर्ड शॉर्टकट उपयोगिता में URL-खोलने की क्रियाएं सेट करें
   उदाहरण: Cmd+Shift+1 → https://extension.tabgroup-trigger/Work
4. टैब समूहों को तुरंत स्विच करने के लिए अपनी कॉन्फ़िगर की गई हॉटकी दबाएं

【कीबोर्ड शॉर्टकट उपयोगिता उदाहरण】
• macOS: BetterTouchTool, Keyboard Maestro
• Windows: AutoHotkey, PowerToys (संस्करण 0.79 या बाद का)

【सेटअप उदाहरण (BetterTouchTool)】
1. एक नया कीबोर्ड शॉर्टकट बनाएं (उदा.: Cmd+Shift+1)
2. क्रिया को "Open URL" पर सेट करें
3. URL दर्ज करें: https://extension.tabgroup-trigger/Work
4. अब Cmd+Shift+1 दबाने से आपके "Work" टैब समूह में तुरंत स्विच हो जाएगा

【सुझाव】
• एक्सटेंशन याद रखता है कि प्रत्येक समूह में कौन सा टैब अंतिम बार सक्रिय था
• जब आप किसी समूह में स्विच करते हैं, तो यह उस टैब को सक्रिय करेगा जिसे आप अंतिम बार देख रहे थे
• यदि कोई टैब इतिहास मौजूद नहीं है, तो यह समूह में पहला टैब सक्रिय करता है

【सीमाएं】
• सहेजे गए टैब समूहों में स्विच नहीं कर सकता (Chrome की "समूह सहेजें" सुविधा जो सभी टैब हटा देती है लेकिन समूह लेबल रखती है)
  नोट: संकुचित टैब समूह (जहां टैब छिपे हुए हैं लेकिन हटाए नहीं गए हैं) ठीक काम करते हैं
• समूह नामों में विशेष वर्ण (?, /, #, %, आदि) को URL एन्कोडिंग की आवश्यकता होती है
  उदाहरण: "Question?Answer" → https://extension.tabgroup-trigger/Question%3FAnswer
  स्वचालित रूप से URL-एन्कोडेड URL प्राप्त करने के लिए एक्सटेंशन के पॉपअप UI का उपयोग करें
• यदि आप 25+ लगातार टैब समूह स्विच करते हैं, तो Chrome की सत्र इतिहास सीमा (chrome.sessions.MAX_SESSION_RESULTS = 25) के कारण पुराने टैब Cmd+Shift+T के माध्यम से पुनर्स्थापित नहीं हो सकते हैं
```
